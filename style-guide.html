Overall Formatting

The website uses an external style sheet called format.css stored in the _css folder. This file controls the main look of the portfolio.

The overall site colors use a light background with darker text so the page is easy to read. The general font-family for the site is Arial with sans-serif as a backup font.

The body style controls the page background color and the default font for the site.

Site Header / Branding

The main heading style changes the color of the portfolio title and centers it on the page. It also makes the heading larger so it stands out more.

One embedded style changes the spacing between the letters in the main heading.

Navigation Links

The navigation styles change the background color of the navigation area and add spacing around the menu.

The list style removes the bullet points from the navigation links.

The navigation list items are displayed in a line so the menu looks cleaner.

The link style changes the text color and removes the underline from the navigation links.

A hover style changes the color of the links when the mouse moves over them.

One embedded style changes the spacing above and below the navigation list.

Content Section

The paragraph style changes the text size and text color so the main content is easier to read.

One embedded style changes the line height of paragraphs to improve readability.

One inline style makes a navigation link bold.

One inline style changes the color of another navigation link.

One inline style adds a background color to the main paragraph on the homepage.

Extra Style Rules from format.css

Additional CSS rules are used to control spacing such as padding, margin, and alignment. These rules help the layout look cleaner and more organized.

These styles work together to give the portfolio a simple, clear, and more polished design.

Style Guide Purpose

This style guide explains the formatting choices used in the portfolio website. It will be updated throughout the quarter as more styles are added to the site.
