Overall Formatting

The website uses an external style sheet called format.css stored in the _css folder. This file controls most of the main styling for the portfolio.

The main font used across the site is Arial, sans-serif. This font was chosen because it is simple, clean, and easy to read.

The body uses a light background color. In the style sheet, one body rule uses background-color: #f4f8fb; with color: #222222;, and another body rule later uses background-color: #f5f5f5;. Because the later rule appears lower in the style sheet, it overrides the earlier background color. The body also uses margin: 0; and padding: 0; to remove the browser’s default spacing.

Site Header / Branding

The main heading is styled in more than one place. In the external style sheet, the .site-title class uses color: #1d4ed8;, text-align: center;, margin-top: 25px;, and font-size: 2.2em;. This makes the title stand out and keeps it centered.

On the homepage, the embedded style for h1 adds letter-spacing: 2px;. Another rule in the style sheet sets the h1 color to navy and centers it. These styles help create a clean header area.

Navigation Links

The navigation area uses a light blue color scheme. The .main-nav class uses background-color: #dbeafe;, padding: 15px;, border-top: 2px solid #93c5fd;, and border-bottom: 2px solid #93c5fd;.

A second navigation rule for the nav element uses background-color: lightblue; and padding: 10px;. This is part of the simpler styling used directly on the current portfolio page.

The navigation list removes bullet points with list-style-type: none;. The embedded style in index.html also adds margin-top: 10px; and margin-bottom: 10px; to the navigation list for spacing.

Navigation list items are displayed in a row using display: inline-block;. The .nav-item class uses margin: 0 12px;, while the simpler nav li rule uses margin-right: 15px;.

Navigation links remove underlines with text-decoration: none;. The .nav-link class uses color: #1e3a8a; and font-weight: bold;. Another simpler rule uses color: black;. On hover, links change color to #dc2626 or red to show user interaction.

Paragraph and Content Styling

Paragraph text is styled for readability. The external rule for p uses font-size: 18px; and color: #333333;. The embedded style in the homepage adds line-height: 1.6; to make the text easier to read.

The homepage also uses inline styles on paragraphs. The background color #f0f8ff is added directly to the paragraph boxes. This gives the content a soft highlighted area.

The .intro-text class includes text-align: center;, font-size: 1.1em;, margin: 30px;, padding: 15px;, background-color: #ffffff;, border: 1px solid #cbd5e1;, and border-radius: 10px;. These values create a centered content box with spacing and rounded corners.

Highlight and Layout Styles

The .highlight-box class is used for a wider content section. It uses width: 80%; to control layout size, margin: 20px auto; to center the box, and padding: 15px; for inside spacing.

This class also uses background-color: #e0f2fe; and border-left: 6px solid #0284c7; to create a highlighted section that stands out. These measurements make it easier for someone to extend the page design later.

Footer and Extra Link Styles

The .footer-note class uses text-align: center;, font-size: 0.9em;, color: #555555;, margin-top: 30px;, and margin-bottom: 20px;. This creates a smaller, softer footer style.

The .style-guide-link class uses display: block;, text-align: center;, margin-top: 20px;, and font-weight: bold; to make the style guide link easier to notice.

List Styling

The general ul rule uses padding-left: 20px; to indent lists. The general li rule uses margin-bottom: 5px; to add space between list items.

Inline Styles Used on the Homepage

One inline style makes the Home navigation link bold using font-weight: bold;.

One inline style changes the Link Lab navigation link color to darkblue.

One inline style adds the background color #f0f8ff to the main paragraph on the homepage.

Style Guide Purpose

This style guide explains both what the styles do and the exact values used, including color values, font names, sizing, spacing, and layout measurements. This information would help another person edit, maintain, and extend the portfolio styling later.
