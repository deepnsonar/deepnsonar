Portfolio Style Guide

Overall Site Style

My website uses an external style sheet called format.css stored in the _css folder. This file controls the main look of the portfolio.

The site uses Arial, sans-serif as the main font. I chose this font because it is simple, clean, and easy to read.

The body uses the background color #f4f1ea and the main text color #222222. The body also uses margin: 0; so the page stretches neatly across the browser window.

Header Styling

The header section uses the background color #7a5c45 with white text. It is centered using text-align: center; and has padding: 20px; to give the title more space.

The header h1 rule uses margin: 0; to remove extra space around the heading. An embedded style also adds letter-spacing: 2px; to spread the letters out slightly.

Navigation Styling

The nav section uses the background color #d8c3a5 and padding: 10px; to separate it from the header and main content.

The nav ul rule removes bullet points with list-style-type: none;. It also centers the menu with text-align: center;, removes extra left spacing with padding: 0;, and adds margin-top: 10px; and margin-bottom: 10px;.

The nav li rule uses display: inline-block; and margin: 0 10px; so the navigation links appear in one row with spacing between them.

The nav a rule removes underlines with text-decoration: none;, uses the text color #3e2723, and makes the links bold with font-weight: bold;.

The nav a:hover rule changes the link text to white, changes the background to #7a5c45, adds padding: 4px 8px;, and rounds the corners with border-radius: 4px;.

Main Content Styling

The main section uses a white background so the content stands out clearly. It uses max-width: 900px; to control the width of the content area and margin: 20px auto; to center it on the page.

The main content also uses padding: 20px; for inner spacing, border-radius: 8px; for rounded corners, and box-shadow: 2px 2px 8px #cccccc; for a soft shadow effect.

The main h2 headings use the brown color #7a5c45 to match the header and footer.

Paragraphs use line-height: 1.6; to improve readability and make the text feel less crowded.

Image Styling

Images are styled with max-width: 100%; and height: auto; so they resize correctly and do not overflow outside their section.

The image border uses 3px solid #7a5c45, and the corners are rounded with border-radius: 8px;. This matches the rest of the site style.

The figure element uses text-align: center; so the image and caption are centered.

Footer Styling

The footer uses the same background color as the header, #7a5c45, with white text to keep the site design consistent.

The footer is centered with text-align: center; and uses padding: 15px; for spacing.

Footer links use the color #ffe9c7 and remove underlines with text-decoration: none;. On hover, the link underlines to show interaction.

CSS Selectors Used

My style sheet uses a variety of CSS selectors on both the index.html and about.html pages.

  • Element selectors: body, header, nav, main, footer
  • Descendant selectors: header h1, nav ul, nav li, nav a, main h2, footer a
  • Pseudo-class selectors: nav a:hover and footer a:hover

Embedded and Inline Styles

I also used embedded styles inside the page head. These include letter-spacing: 2px; for the main heading, margin-top: 10px; and margin-bottom: 10px; for the navigation list, and line-height: 1.6; for paragraph text.

I used inline styles on some links and text in my pages so I could practice applying CSS directly inside HTML elements.

Style Guide Purpose

This style guide explains how the portfolio is styled and includes specific information such as color values, font names, sizing, spacing, and layout measurements. This would help another person edit, maintain, and extend the design of the site later.
